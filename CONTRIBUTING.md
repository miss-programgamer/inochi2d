## On Generative AI
Generative AI in the form of large language models, diffusion models, etc. may not be used for contributing
code or assets to the upstream Inochi2D codebase. While it's possible some people may be able to pass code 
review using AI generated code, we will remove and from-scratch rewrite any code submitted that we find was 
made using generative AI. As such, generative AI is **banned** from use in Inochi2D.

Attempts to circumvent this will result in a permanent ban from contributing to the upstream Inochi2D project.

## Contribute code changes
Raise a pull request on our [Github project](https://github.com/Inochi2D/inochi2d) and follow the template.

### Notes
* We do not accept contributions from people who work for/at Live2D inc. and affiliates.
* We will not accept pull requests that aim to support other 2d puppet formats within Inochi2D.
* Please do not submit code that is stolen from propriatary software.
* Please do not submit code written whole, or in part, using generative AI (such as Claude Code, ChatGPT, Copilot, etc.)
